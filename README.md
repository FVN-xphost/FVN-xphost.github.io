# 福瑞视觉小说

首先，在这里先和各位说明一下情况：

1. 本仓库下的所有图片均为透明无水印的立绘，但是这并不意味着所有人均可以无偿拿走，本仓库下的所有图片版权均归其画师/原作者所有，任何人均需要得到他们的授权才能转载或者二次开发。
2. 本仓库的目的仅是为了跨平台编译，并给各位提供一个 release 发布平台，各位当然也可以提出 issue 以让我整改。
3. 本仓库仅用于参与 FVN Game jam 组织旗下活动**言语肇始**，当活动结束后，本仓库即刻遵循原画师以及原剧本的意图进行**存档**。并不再更新！
4. 如何获得立绘原作者的授权？在我的[贡献者名单](./CONTRIBUTOR.md)里面有，各位请自行去私聊他们的 b 站账号。

本仓库目前的许可证是 All rights reserved. 如果没有特殊说明，严禁使用本开源地址进行二次开发！！

4. 【推荐】`macOS aarch64`（`M1`、`M2`）：Actions 下载 macOS aarch64 即可！
5. `macOS x86_64`（`Intel`）：Actions 下载 macOS x86_64 即可！
6. 【推荐】`Linux x86_64`（`Intel`、`AMD`）：Actions 下载 Linux x86_64 即可！
7. `Linux aarch64`（`树莓派`）：Actions 下载 Linux aarch64 即可！
8. 【推荐】`Android aarch64`（`骁龙 8（845、855、865） 以上`、`麒麟`、`鲲鹏`）：Actions 下载 Android aarch64 即可！

10. `Android x86_64`（`ChromeOS`、`Intel`、`Android 模拟器`）：Actions 下载 Android x86_64 即可！
