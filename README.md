# 福瑞视觉小说

首先，在这里先和各位说明一下情况：

1. 本仓库下的所有图片均为透明无水印的立绘，但是这并不意味着所有人均可以无偿拿走，本仓库下的所有图片版权均归其画师/原作者所有，任何人均需要得到他们的授权才能转载或者二次开发。
2. 本仓库的目的仅是为了跨平台编译，并给各位提供一个 release 发布平台，各位当然也可以提出 issue 以让我整改。
3. 本仓库仅用于参与 FVN Game jam 组织旗下活动**言语肇始**，当活动结束后，本仓库即刻遵循原画师以及原剧本的意图进行**存档**。并不再更新！
4. 如何获得立绘原作者的授权？在我的[贡献者名单](./CONTRIBUTOR.md)里面有，各位请自行去私聊他们的 b 站账号。
5. 在线游玩网址：[点我](https://FVN-xphost.github.io)

本仓库目前的许可证是 All rights reserved. 如果没有特殊说明，严禁使用本开源地址进行二次开发！！
